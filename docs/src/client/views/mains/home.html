What is rapid-build?

rapid-build is an npm package.

rapid-build is a build system utilizing Node.js, Express and AngularJS.

rapid-build allows you to build awesome things fast with the latest and greatest technologies.

Install rapid-build by running...

$ npm install rapid-build # or $ npm install rapid-build -g

See getting started for installation help.
