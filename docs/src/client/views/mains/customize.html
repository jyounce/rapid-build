Custom Build Options

If the default builds don't meet your needs, no problem!
Below are a list of options you can use to customize your builds.

browser

  • open (boolean - true)

    description
    • Open the browser once the build completes, applies to builds: default, dev and prod:server.
    example
    var opts = { browser: { open: false } }

  • reload (boolean - true)

    description
    • Reloads the browser once you save your src file, only applies to dev builds.
    example
    var opts = { browser: { reload: false } }

build

  • client | server (boolean - true)

    description
    • Set to false to skip building the client or server.
    • Atleast one is required to be true.
    example
    var opts = { build: { server: false } }
